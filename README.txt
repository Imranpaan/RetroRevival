1. PROJECT OVERVIEW
Retro Revival is an online e-commerce platform dedicated to thrifted and vintage clothing. 

2. INSTALLATION AND SETUP INSTRUCTIONS
- Ensure you have XAMPP installed on your Windows machine.
- Extract the submitted "RetroRevival" folder.
- Move the entire "RetroRevival" folder into your XAMPP directory path: C:\xampp\htdocs\
- Open the XAMPP Control Panel and click "Start" for both the Apache and MySQL modules.

3. DATABASE CONFIGURATION STEPS
- Open your web browser and go to: http://localhost/phpmyadmin/
- Click on "New" in the left sidebar to create a new database.
- Name the database exactly: retro_revival (and click Create).
- Select the newly created "retro_revival" database and click the "Go" tab at the bottom.
- Click "Choose File" and select the "retro_revival.sql" file included in this submission folder.
- Scroll down and click "Import" to populate the tables and dummy data.

4. SYSTEM ACCESS AND LOGIN CREDENTIALS
Once the database is imported, you can access the system by visiting:
http://localhost/RetroRevival/index.php
